⌨️ 입력 방법
📌 단원정보 — 상단 바에 단원명·출처·페이지·문제번호 입력.
📦 각 박스 — 박스 우측 상단에 쪽·문제번호 개별 입력.
✏️ 텍스트 풀이 — 박스 내부 클릭 후 바로 입력.
🖊 손글씨 — 상단 펜/형광/지우개로 모눈 위에 직접 필기.
🎨 색상·굵기 — 컬러 팔레트와 두께 버튼으로 조절.
🧠 효과적인 풀이 정리 4원칙
1
단계별 번호 — ①②③ 으로 흐름을 끊어 쓰면 검토가 쉬움.
2
조건·공식을 먼저 — 풀기 전 주어진 조건과 쓸 공식을 박스 상단에 메모.
3
검산 표시 — 풀이 완료 후 ✓ 표시. 의심되는 줄은 빨간 펜으로 체크.
4
틀리면 오답노트로 — 이 노트에서 틀린 문제는 오답노트에 다시 정리.
💾 저장 & 출력
🖼 이미지저장 — 페이지별 PNG 고해상도. 카톡 공유 가능.
📄 PDF저장 — 전 페이지 A4 PDF 한 파일.
🖨 인쇄 — 여백 "없음" 설정 시 한 장에 딱 맞게 출력.
📲 홈화면추가 — GitHub Pages 업로드 후 PWA로 설치.
📐 모눈 칸 활용법
📏 5mm 모눈 — 수식·도형·그래프를 정렬감 있게 작성.
↕ 칸 나누기 — 가로선을 긋거나 펜으로 세로 구분선 추가.
🎨 색상 구분 — 파랑:풀이 / 빨강:실수·검산 / 초록:공식.
1페이지 = 4문제 — 한 장에 4문제씩 체계적으로 관리.
