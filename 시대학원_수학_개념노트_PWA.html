⑤ 위의 홈화면 추가 방법으로 설치 ✅

📝 텍스트 입력
각 칸을 클릭하면 바로 타이핑 가능. 수식은 기호(√, ∞, ≤)를 복사해서 붙여넣기.
🖼 이미지 첨부
문제란 상단 📁 이미지 버튼으로 파일 선택, 또는 이미지를 문제란으로 드래그. 모바일에서는 카메라 촬영도 가능.
📋 클립보드 붙여넣기
문제란 클릭 후 Ctrl+V (PC) 또는 길게 누르기→붙여넣기 (모바일)로 캡처 이미지 삽입.
🖊 손글씨·필기 도구
펜 모드 — 상단 🖊 버튼 활성화 후 화면에 직접 필기. 스타일러스/손가락 모두 지원.
형광펜 — 🖍 버튼으로 반투명 형광 표시. 중요 개념이나 공식에 활용.
지우개 — ◻ 버튼으로 손글씨 부분 삭제. ✕로 전체 삭제.
색상·굵기 — 색상 점(🔵🔴⚫🟢🟣)과 굵기 버튼으로 조절.
💡 손글씨 모드 켜면 텍스트 편집이 잠깁니다. 다시 버튼 누르면 해제.

💾 저장 & 출력 방법
🖼 이미지 저장 — PNG 고해상도로 페이지별 저장. 카톡·메모앱에 바로 공유 가능.
📄 PDF 저장 — 전체 페이지를 A4 PDF 한 파일로. 프린트·디지털 보관용.
🖨 인쇄 — 브라우저 인쇄 대화상자. 여백을 없음으로 설정하면 한 장에 딱 맞게 출력.
💡 인쇄 전 배경 그래픽 인쇄 옵션 체크하면 줄선이 출력됩니다.
